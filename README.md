# Gadget Shop Website 🛒

## Project Description

Gadget Shop is a simple e-commerce website that allows users to browse electronic gadgets and add them to a shopping cart. The project demonstrates basic web development concepts using HTML, CSS, Bootstrap, and JavaScript.

You can Access the website here: https://taupe-alfajores-52158f.netlify.app/

## Features

* Browse different gadgets such as smartphones, laptops, TVs, speakers, smartwatches, and power banks
* Add products to the shopping cart
* View selected products in the cart page
* Simple and responsive layout using Bootstrap
* Navigation bar with Home, Login, and Cart pages

## Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **Bootstrap 5** – Responsive design and UI components
* **JavaScript** – Cart functionality and data storage

## Project Structure

```
GadgetShop/
│
├── index.html        # Main page displaying gadgets
├── Cart.html         # Shopping cart page
├── login.html        # Login page
├── style.css         # Website styling
├── cart.js           # JavaScript for cart functionality
└── pics/             # Images used for products
```

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Run **index.html** in any web browser.
4. Click **Add to Cart** to add products and open the **Cart page** to view them.

## Purpose of the Project

This project was created to practice front-end web development and understand how a basic shopping cart system works in an e-commerce website.
